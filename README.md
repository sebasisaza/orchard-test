# My Angular App

Angular application for Orchard

## Prerequisites

- Node.js (v14+)
- npm (v6+)
- Angular CLI (v14+)

## Installation

1. Clone the repo:
   ```bash
   git clone https://github.com/sebasisaza/orchard-test.git
   cd orchard-test/
   ```
2. ```console
   npm install
   ```

3. ```console
   npm start
   ```
